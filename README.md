# Student Data Analyzer

A menu-driven C++ application that stores and manages student records using arrays. The project demonstrates the implementation of searching and sorting algorithms as part of Data Structures and Algorithms (DSA).

## Features

- Add student records
- Display all student records
- Linear Search by Roll Number
- Linear Search by Name
- Binary Search by Roll Number
- Selection Sort by Roll Number (Ascending)
- Bubble Sort by Marks (Descending)
- Input validation for roll number and marks
- Prevents duplicate roll numbers
- Supports names with spaces
